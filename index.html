CourtMatch 隱私

隱私政策

版本 1.1　·　最後更新 2026 年 7 月 14 日　·　生效日期 2026 年 7 月 14 日

CourtMatch 重視您的個人資料私隱。本《隱私政策》說明我們在您下載、註冊、登入、瀏覽或使用 CourtMatch 應用程式、網站及相關服務時,如何收集、持有、處理、使用、披露、保護及刪除您的個人資料。

本政策應與《CourtMatch 服務條款》一併閱讀。

請您在提供個人資料或使用本服務前仔細閱讀本政策。對於需要另行取得權限或同意的功能,例如位置、相機、相簿、通知或直接促銷,我們會在相關功能啟用時另行向您作出提示。

✦重點摘要

  • 我們僅收集營運網球約球服務所合理必要的資料:帳號資料、球技水平與偏好、活動紀錄及技術資料。
  • 我們不會出售您的個人資料,亦不會向第三方提供您的個人資料供其自身直接促銷之用。
  • 位置、相機、相簿及通知僅在您授權後使用,您可隨時在裝置設定中撤回權限。
  • 您可以查閱、更正、刪除資料,並可隨時在應用程式內刪除帳號。
  • 本服務僅供年滿 14 歲人士使用;未滿 18 歲用戶須經家長或監護人同意,並適用額外保護措施。

本摘要僅為方便閱讀,具體以下文完整政策為準。

2本政策的適用範圍

本政策適用於:

  1. CourtMatch iOS、Android 或其他應用程式;
  2. CourtMatch 官方網站及網頁版服務;
  3. CourtMatch 提供的約球、球友配對、聊天、賽事、評價、信譽、通知及客服功能;

本政策不適用於由第三方獨立營運的網站、應用程式或服務。第三方將依其自身的隱私政策處理您的資料。

3我們收集的個人資料

我們只會在提供服務、保障安全或實現下述目的所合理需要的範圍內收集個人資料。實際收集的資料取決於您使用的功能。

3.1 帳號及身份資料

我們可能收集:

  1. 用戶名及顯示名稱;
  2. 手機號碼或電郵地址;
  3. 登入及驗證資料;
  4. 出生日期及據此產生的年齡或年齡組別;
  5. 性別或您選擇公開的性別資訊;
  6. 頭像及個人簡介;
  7. 居住地區、常用語言及時區;
  8. 帳號建立、驗證、停用及刪除狀態;
  9. 在需要時用於核實帳號或處理申訴的身份證明資料。

除非我們另行說明,手機號碼或電郵地址、出生日期、用戶名以及完成註冊所必需的資料通常屬必須提供的資料。如您不提供,可能無法建立帳號或使用部分核心功能。

就上述第 9 項收集的身份證明材料,我們僅用於相關核實或申訴,並在目的達成後刪除或作遮蓋處理,但法律要求或爭議處理需要繼續保留者除外。

3.2 網球及活動資料

我們可能收集:

  1. NTRP、UTR、自訂水平或其他網球能力資料;
  2. 單打、雙打、比賽制式及球友偏好;
  3. 常去球場、偏好地區及可用時間;
  4. 約球發布、申請、接受、拒絕、取消及出席紀錄;
  5. 比賽、比分、排名、分組、賽程及賽事紀錄;
  6. 評價、信譽分、爽約紀錄及相關申訴;
  7. 關注、收藏、封鎖及配對紀錄。

除出生日期和其他非公開資料外,部分網球及活動資料可能依您的設定向其他用戶公開,以便提供配對、活動和社群功能。

3.3 用戶產生內容及通訊資料

我們可能處理:

  1. 私信、群組聊天及活動聊天室內容;
  2. 帖文、留言、評價、圖片、影片及其他上傳內容;
  3. 您發送給客服的訊息及附件;
  4. 舉報、投訴、申訴、封鎖及內容審核紀錄;
  5. 其他用戶在合法使用服務時提交並涉及您的內容,例如對您的評價、比賽紀錄或舉報。

我們不會把私人聊天內容公開展示,但可能在處理舉報、保障安全、調查違規、遵從法律要求或維護服務時,由獲授權人員或服務供應商在必要範圍內查閱。

請勿透過公開個人檔案、帖子、聊天室或比賽資料公開您不希望他人知悉的敏感資料。

3.4 位置及球場資料

在您授權後,我們可能收集裝置提供的大概位置或精確位置,用於:

  1. 顯示附近球場、活動或球友;
  2. 計算距離及排序搜尋結果;
  3. 自動填入活動地區;
  4. 提供與位置有關的安全或服務功能。

除非另有明確說明,本應用不會在您未使用相關功能時持續於背景追蹤您的位置。

您可透過裝置設定撤回位置權限,但附近球場、距離排序及其他位置功能可能因此無法正常使用。

3.5 裝置、網絡及技術資料

我們可能自動收集:

  1. 裝置型號、作業系統及應用程式版本;
  2. IP 地址、網絡類型、語言、時區及大概地區;
  3. 裝置或安裝識別資料;
  4. 登入時間、操作紀錄及安全日誌;
  5. 崩潰報告、錯誤訊息及性能資料;
  6. 推送通知令牌;
  7. 防止濫用及保障帳號安全所需的技術訊號。

我們不會僅為廣告目的建立裝置指紋,亦不會在未依法取得所需授權的情況下,將您的資料與第三方資料結合以跨應用程式或網站追蹤您。

3.6 相機、相簿及媒體資料

當您選擇上傳頭像、聊天圖片、比賽照片或其他內容時,我們可能請求存取:

  1. 相機;
  2. 您選擇的相片或影片;
  3. 相關媒體檔案資料。

我們只會在您主動使用相關功能並授權後存取上述資料。您可在裝置設定中撤回權限。

3.7 通知及通訊偏好

如您開啟通知,我們可能處理推送通知令牌及通知偏好,用於發送:

  1. 約球申請、接受、取消及時間提醒;
  2. 聊天訊息;
  3. 賽事及比分更新;
  4. 帳號、安全或服務公告;
  5. 您選擇接收的產品更新或推廣資訊。

您可在應用程式或裝置設定中調整通知偏好。部分必要的帳號、安全或法律通知可能不能完全關閉。

3.8 付款及交易資料

如本服務提供 App 內購買、訂閱、報名費或其他付費功能,付款一般由 Apple、Google 或其他付款服務供應商處理。

我們通常不直接取得您的完整銀行卡資料,但可能收到:

  1. 交易識別碼;
  2. 購買項目及金額;
  3. 交易狀態;
  4. 訂閱、退款或恢復購買狀態;
  5. 處理客服、爭議及財務核對所需的紀錄。

3.9 監護人資料

如未成年用戶須經家長或法定監護人同意,我們可能收集監護人的:

  1. 姓名;
  2. 電郵地址或手機號碼;
  3. 與未成年人的關係;
  4. 同意、撤回同意及聯絡紀錄;
  5. 核實監護人身份所合理需要的有限資料。

我們只會將監護人資料用於未成年人保護、取得或記錄同意、處理安全事項及回應權利請求。

4我們如何收集資料

我們可能透過以下方式收集個人資料:

  1. 您註冊、建立或編輯帳號時直接提供;
  2. 您發布約球、參加比賽、聊天或使用其他功能時產生;
  3. 您允許裝置權限後由裝置提供;
  4. 您與客服聯絡、舉報或申訴時提供;
  5. 其他用戶在使用評價、賽事、聊天或舉報功能時提交;
  6. 我們的伺服器、應用程式及安全系統自動記錄;
  7. 經您授權後,由第三方登入或其他合作服務提供;
  8. 由付款平台、推送服務、地圖服務或其他服務供應商提供交易或服務狀態。

5我們使用個人資料的目的

我們可能將個人資料用於以下目的:

5.1 提供及管理服務

  1. 建立、驗證及管理帳號;
  2. 提供約球、搜尋、配對、聊天及賽事功能;
  3. 展示個人檔案、活動、比分及公開內容;
  4. 處理活動申請、取消、出席及評價;
  5. 提供球場推薦及位置相關功能;
  6. 處理付款、訂閱、退款及客戶支援;
  7. 向您發送服務通知。

5.2 個人化及推薦

我們可能根據您的球技水平、地區、時間、活動偏好、互動及使用紀錄,向您推薦較相關的球友、活動、球場或賽事。

除非另有明確說明,這些推薦不會對您產生法律或類似重大影響。如我們日後引入可能產生該等影響的自動化決定,我們會在其適用於您之前通知您,並提供法律要求的選擇。

5.3 信譽、評價及平台治理

我們可能使用出席、取消、評價、舉報及活動紀錄:

  1. 計算或調整信譽分;
  2. 顯示出席或活動表現;
  3. 識別惡意爽約、欺詐、騷擾或濫用;
  4. 執行社群規範和服務條款;
  5. 處理舉報、申訴及帳號限制。

5.4 安全、反欺詐及法律遵從

我們可能處理資料,以:

  1. 防止未經授權登入及帳號盜用;
  2. 偵測垃圾訊息、機械人、詐騙及濫用;
  3. 保護未成年人及其他用戶安全;
  4. 調查投訴及違規;
  5. 保全證據及處理法律爭議;
  6. 回應有效的法律程序或政府要求;
  7. 保護 CourtMatch、用戶或公眾的合法權益及安全。

5.5 分析及改善服務

我們可能使用彙總、去識別化或必要的使用資料:

  1. 分析功能使用情況;
  2. 修復錯誤及改善性能;
  3. 評估新功能;
  4. 進行容量、安全及可靠性規劃。

在合理可行情況下,我們會優先使用無法直接識別個人的資料。

5.6 推廣及直接促銷

除服務、安全及交易通知外,我們僅會按照《條例》第 6A 部的規定將您的個人資料用於直接促銷。具體而言:

  1. 資料種類:我們可能用於直接促銷的個人資料僅限於您的姓名、聯絡方式(電郵、手機號碼或推送令牌)、地區、語言偏好、球技水平、活動偏好及服務使用紀錄。
  2. 促銷標的類別:有關促銷僅涉及 CourtMatch 的功能、活動、賽事、網球相關活動,以及與網球相關的產品或服務。
  3. 同意先行:我們會事先告知上述事項,並在取得您的同意或您表示不反對(按法律規定)之前,不會將您的個人資料用於直接促銷。
  4. 不作第三方促銷:未經您書面同意,我們不會向任何其他人士提供您的個人資料,供其用於自身的直接促銷。
  5. 免費退出:您可隨時透過取消訂閱方式、應用程式設定或聯絡我們,要求停止將您的個人資料用於直接促銷,我們將免費執行。

6個人檔案及公開資料

為提供約球和社群功能,以下資料可能依您的設定向其他用戶顯示:

  1. 用戶名、顯示名稱及頭像;
  2. 自我介紹;
  3. 年齡組別,而非完整出生日期;
  4. 性別或您選擇公開的性別資料;
  5. NTRP、UTR 或其他球技水平;
  6. 常用地區、偏好球場及活動偏好;
  7. 約球、比賽、比分、排名及出席資料;
  8. 評價、信譽分或相關平台標記;
  9. 您主動發布的帖子、圖片及其他公開內容。

您的手機號碼、電郵地址、完整出生日期、精確即時位置及監護人聯絡方式不會作為一般公開個人檔案內容展示,除非您自行公開或明確選擇分享。

即使您刪除公開內容,其他用戶仍可能已截圖、轉發或保存該內容。我們無法控制其他用戶在本服務以外的行為,但可依規則處理違規披露或濫用。

7我們向誰披露或轉移資料

我們不會出售您的個人資料,亦不會向任何第三方提供您的個人資料,供其用於該第三方自身的直接促銷。

我們可能在實現本政策所述目的的必要範圍內,向以下類別的接收方披露或轉移資料:

7.1 其他用戶

為提供個人檔案、約球、賽事、聊天、評價及社群功能,我們會依您的設定向其他用戶顯示相關資料。

7.2 服務供應商及資料處理者

我們可能委託以下類別的供應商代為處理資料:

  1. 雲端運算、伺服器及資料庫供應商;
  2. 圖片、影片及檔案儲存供應商;
  3. 短訊、電郵及身份驗證供應商;
  4. 推送通知服務;
  5. 地圖、地理編碼、球場及天氣資料服務;
  6. 崩潰分析、日誌、性能及安全服務;
  7. 客服、內容審核及反欺詐服務;
  8. 支付、訂閱及財務服務;
  9. 專業顧問、核數師、保險公司及法律顧問。

我們會以合約或其他方法,要求代表我們處理資料的服務供應商只按我們的指示及適用目的處理資料,採取適當的保密及安全措施,並且不得超期保留資料。

7.3 法律、監管及安全需要

在合理相信有必要時,我們可能向法院、執法機關、監管機構、緊急服務或其他有權機構披露資料,以:

  1. 遵守法律、法院命令或有效法律程序;
  2. 調查犯罪、欺詐、騷擾或安全事件;
  3. 防止對任何人的生命、身體或財產造成重大風險;
  4. 保護 CourtMatch、用戶或公眾的合法權益。

7.4 業務轉讓

如 CourtMatch 發生合併、融資、重組、資產出售或業務轉讓,個人資料可能作為相關業務的一部分移交給實際或潛在交易方,但接收方須受適當的保密及資料保障要求約束。如該等轉讓實質改變您個人資料的使用方式,我們將按第 17 條通知您。

8跨境儲存及處理

我們及服務供應商可能在香港以外的地區儲存或處理個人資料。相關地區的資料保護法律可能與香港不同。

如發生跨境處理,我們將在合理可行情況下採取適當措施,包括:

  1. 評估服務供應商的安全及私隱措施;
  2. 透過合約限制資料的用途,並在適當情況下參考香港個人資料私隱專員公署發布的《建議範本合約條款》;
  3. 限制人員存取;
  4. 使用傳輸加密及其他安全措施;
  5. 僅轉移實現有關目的所需的資料。

9資料保存期限

我們只會在實現收集目的、提供服務、履行法律義務、保障安全或處理爭議所需的期間保存個人資料。我們設有內部保存期限安排,並定期檢視所儲存的資料。

一般而言:

  1. 帳號及個人檔案資料:保存至帳號刪除或不再需要為止;
  2. 約球、比賽及評價紀錄:在帳號存續期間保存,刪除帳號後可能保留去識別化紀錄或為保障其他用戶權益所需的有限資料;
  3. 聊天內容:保存至用戶刪除、相關會話或帳號依規則清理,或不再需要為止;
  4. 安全、登入及反欺詐紀錄:依風險及安全需要保存合理期間;
  5. 客服、舉報及申訴紀錄:保存至個案完結後的合理期間,以處理後續爭議及保障平台安全;
  6. 交易及付款紀錄:依法例、會計、稅務、退款及爭議處理需要保存;
  7. 系統備份:可能在正常備份輪換週期內暫時保留,並在輪換完成後刪除或覆寫。

當資料不再需要時,我們會採取合理措施刪除、匿名化或安全隔離該資料。

帳號刪除不一定導致所有資料即時及無條件刪除。如法律要求、未完成交易、欺詐防範、安全事件、投訴、訴訟、執法要求或保護其他用戶權益需要,我們可能保留必要的最少資料。

10資料安全

  1. 傳輸加密;
  2. 密碼雜湊及安全驗證;
  3. 存取控制及最小權限;
  4. 日誌、監控及異常偵測;
  5. 備份及災難復原措施;
  6. 服務供應商安全評估;
  7. 員工及承辦商保密要求;
  8. 安全測試、漏洞修補及事件處理程序。

然而,任何互聯網傳輸、電子儲存或資訊系統均不能保證絕對安全。您亦應使用安全的登入方式,不向他人披露驗證碼,並在發現帳號異常時立即聯絡我們。

11資料外洩及安全事件

如發生實際或懷疑的個人資料外洩,我們將按事件性質採取合理措施,包括:

  1. 控制及停止外洩;
  2. 調查原因及影響;
  3. 修補安全漏洞;
  4. 評估對用戶造成的風險;
  5. 如外洩對受影響用戶構成實際損害風險,盡快通知受影響用戶及香港個人資料私隱專員公署,並在適當或法律要求的情況下通知執法機關或其他監管機構;
  6. 採取措施防止類似事件再次發生。

13未成年人私隱

13.1 最低年齡

CourtMatch 僅供年滿 14 歲的人士使用。未滿 14 歲者不得註冊或使用本服務。我們不會有意收集未滿 14 歲人士的個人資料;如發現該等資料,我們將依第 13.6 條處理及刪除。

13.2 監護人同意

14 歲至未滿 18 歲的用戶須在家長或法定監護人閱讀並同意本政策及《服務條款》後方可使用本服務。

我們可能要求未成年用戶或監護人完成合理的同意、確認或驗證程序。

13.3 資料最小化

我們會盡量減少收集未成年人的資料。除非提供服務或保障安全所必要,我們不鼓勵未成年人公開:

  1. 完整出生日期;
  2. 電話號碼或電郵地址;
  3. 學校、住址或精確位置;
  4. 身份證明資料;
  5. 可識別日常行程的資訊。

13.4 年齡組別顯示

未成年人的完整出生日期不會向一般用戶公開。其他用戶通常只會看到年齡組別或為安全配對所必要的有限年齡資訊。

13.5 互動限制

CourtMatch 可能依未成年人保護規則,限制未成年用戶與成年用戶之間的搜尋、推薦、約球、私信、關注、群組或賽事互動。

具體限制以應用程式當時實際提供的功能及安全設定為準。我們可能因安全風險、法律要求或產品改進調整有關限制。

13.6 發現不符合資格的帳號

如我們有合理理由相信用戶未達最低年齡,或未依要求取得監護人同意,我們可能:

  1. 限制或暫停帳號;
  2. 要求年齡或監護人同意證明;
  3. 刪除或匿名化相關資料;
  4. 保留處理安全、欺詐、投訴或法律義務所必需的最少資料。

13.7 監護人的權利

家長或法定監護人如認為未成年人未經適當同意提供資料,或希望代表未成年人提出查閱、更正、刪除或安全投訴,可透過第 15 條聯絡我們。

我們可能要求提供合理資料以核實監護人身份及其與未成年人的關係。
